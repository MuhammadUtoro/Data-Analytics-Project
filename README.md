# 📊 Data Analytics Projects Portfolio

This repository is a collection of **end-to-end data analytics projects** built to demonstrate practical skills in data analysis, data cleaning, exploratory data analysis (EDA), and data visualization.

Each project follows a structured workflow from raw data to actionable business insights.

---

## 🧰 Tools & Technologies

The following tools are used across projects:

- **Excel** – Data cleaning, basic analysis, pivot tables
- **Python** – Data cleaning, EDA, and analysis  
  - Pandas  
  - NumPy  
  - Matplotlib  
  - Seaborn  
- **SQL (PostgreSQL)** – Data querying and transformation  
- **Tableau** – Data visualization and dashboard creation  

---

## 📌 Projects Included

This repository will include multiple data analytics projects such as:
